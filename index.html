    I’ve always wanted to create my own website—to create a medium that is uniquely mine through which I can record and broadcast my thoughts and voice. As the world slows down in response to the pandemic, I’ve found time to learn computer science and programming. With some newfound knowledge of HTML, CSS, and JavaScript, I finally have the tools to create my own website from scratch. Here’s to using this platform for writing and developing my programming skills!
